# BizYAML for Visual Studio Code

**Enterprise-grade Domain Specific Language (DSL) for Entity-Driven Development.**

BizYAML is a powerful toolkit designed to standardize business logic, entity structures, and workflows. This extension brings the full power of the BizYAML ecosystem directly into your editor with real-time semantic diagnostics and intelligent automation.

## 🚀 Key Features

- **Semantic Validation (LSP)**: Real-time checks for cross-entity relations, workflow states, and view column existence.
- **Instant Feedback**: Catch logical errors as you type, with clear diagnostics in the Problems panel.
- **Multi-File Intelligence**: Automatically scans your workspace to ensure consistency between `.entity.yaml`, `.flow.yaml`, and `.views.yaml`.
- **Schema Support**: Out-of-the-box JSON Schema validation for the entire YAML structure.

## 📦 Installation

This extension is available on:
- **[VS Code Marketplace](https://marketplace.visualstudio.com/)**
- **[Open VSX Registry](https://open-vsx.org/)**

## 📖 Quick Start

1. Open a workspace containing BizYAML files.
2. Ensure your files follow the naming convention: `*.entity.yaml`, `*.flow.yaml`, or `*.views.yaml`.
3. Start editing! The BizYAML Language Server will automatically monitor your workspace.
